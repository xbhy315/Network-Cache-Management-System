**RESP 模式 getAllEntries 实现参考**：

```java
private List<CacheEntry> getAllEntries() {
    if (client instanceof MockCacheClient mock) {
        return mock.getAllLocalEntries();
    }
    // RESP 模式：SCAN 获取所有 key → 逐个 GET 获取值
    List<String> keys = client.scan("*");  // 等待接口放开
    List<CacheEntry> entries = new ArrayList<>();
    for (String key : keys) {
        // 跳过 LIST 类型 key（无法通过 GET 获取）
        // 先尝试 GET，非 null 则为 STRING 类型
        client.get(key).ifPresent(value -> 
            entries.add(new CacheEntry(key, EntryType.STRING, value, ...)));
    }
    return entries;
}
```
